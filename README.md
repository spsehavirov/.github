# SPŠE Havířov `.github`

Tento repozitář slouží pro nastavení a obsah na úrovni celé organizace `spsehavirov`.

## Co je zde důležité

- [`profile/README.md`](./profile/README.md) se zobrazuje na hlavní stránce organizace GitHub.
- `README.md` v kořeni tohoto repozitáře popisuje účel samotného repozitáře `.github`.
- Soubory jako `CONTRIBUTING.md`, `SUPPORT.md`, šablony issues nebo PR se mohou použít jako výchozí napříč repozitáři organizace.

## Doporučené využití

V tomto repozitáři má smysl udržovat jen obsah, který:

- platí pro více repozitářů současně,
- pomáhá návštěvníkům zorientovat se v organizaci,
- snižuje duplicitní nastavování v jednotlivých projektech.
